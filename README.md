# 🌤️ 中国区天气显示 - GNOME Shell 扩展

一个专为中国用户设计的 GNOME Shell 天气显示扩展，使用和风天气数据源，支持 ACG 角色系统。

## ✨ 特点

- 🇨🇳 **中国区优化** - 使用和风天气 API，中国区数据最准确
- 🔍 **城市搜索** - 在设置里搜索选择城市，无需手动输入 ID
- 🎭 **ACG 角色系统** - 9 个预置角色，每个有独特性格和台词
- 🌡️ **详细信息** - 温度、湿度、风向、气压、能见度一应俱全
- 📅 **未来预报** - 显示未来 3 天天气预报
- 🌬️ **空气质量** - AQI 指数显示（需要 API 权限）
- 📋 **生活指数** - 穿衣、运动、紫外线等建议
- ⚙️ **可配置** - 角色切换、城市选择、API 配置

## 🚀 安装

### 前置条件

1. **GNOME Shell 42+**（Manjaro GNOME 默认已安装）
2. **和风天气 API Key**（免费注册获取）

### 获取 API Key

1. 访问 [和风天气开发平台](https://dev.qweather.com)
2. 注册账号并登录
3. 创建项目，获取 API Key 和 API Host
4. 免费版每天 1000 次调用，足够个人使用

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/amanannn/weather-cn.git
cd weather-cn

# 运行安装脚本
chmod +x install.sh
./install.sh

# 启用扩展
gnome-extensions enable weather-cn@amanannn
```

## ⚙️ 配置

安装后，点击面板上的天气图标，选择 **⚙️ 设置**。

### 设置选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| API Key | 和风天气 API 密钥 | （必填）|
| API Host | 个人 API Host 地址 | （必填）|
| 城市 | 搜索选择城市 | 北京 |
| 角色 | 选择 ACG 角色 | 元气少女 |

## 🎨 自定义角色

在 `characters/` 目录下创建 JSON 文件即可添加新角色：

```json
{
  "id": "mychar",
  "label": "我的角色",
  "avatar": "mychar.jpg",
  "kaomoji": {
    "sunny": ["(◕‿◕)"],
    "default": ["(─‿─)"]
  },
  "lines": {
    "current": {
      "sunny": ["今天天气不错！"],
      "default": ["今天{temp}度，{condition}。"]
    },
    "forecast": ["{day}：{condition}，{tempMin}～{tempMax}°C。"],
    "indices": {
      "3": "穿衣：{category}。{text}"
    }
  }
}
```

## 🗑️ 卸载

```bash
chmod +x uninstall.sh
./uninstall.sh
```

## 🔧 开发

### 项目结构

```
weather-cn/
├── metadata.json          # 扩展元数据
├── extension.js           # 主要逻辑
├── prefs.js               # 设置页面
├── stylesheet.css         # 样式表
├── characters/            # ACG 角色配置
│   ├── genki.json         # 元气少女
│   ├── genki.jpg          # 元气少女头像
│   └── ...
├── schemas/               # GSettings Schema
├── install.sh             # 安装脚本
├── uninstall.sh           # 卸载脚本
└── README.md              # 说明文档
```

### 技术栈

- **GJS** (GNOME JavaScript) - 扩展开发语言
- **GTK4 / libadwaita** - 设置界面
- **Soup 3** - HTTP 请求
- **和风天气 API v7** - 天气数据源

## 📝 注意事项

1. **API 调用限制**：免费版每天 1000 次，建议刷新间隔设为 30 分钟
2. **网络要求**：需要网络连接访问和风天气 API
3. **GNOME 版本**：支持 GNOME Shell 42-46
4. **空气质量**：需要和风天气 API 特定权限，免费版可能无法使用
